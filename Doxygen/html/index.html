Summary

In this project we have made a virtual environment in which the players can join over the network and will have their avatars which can be identified by their Player ID. These players can add their name so that they can be easily recognized by the other player.

In the game, the players can shoot other player with the controller and same can be visualized with the health bar, as when a player is being shot, the health reduces.

Each avatar has his hand models available whose IK has been implemented to follow the controller of the HTC Vive or Oculus Rift or the Leap motion hand model.

Leap motion hand model has also been enabled for this game which, when enabled will track your hand showing you the bone model and not the avatar’s IK hand and the same can be seen over the network on avatar’s IK hands. The leap hand model also enables them to interact with the ball and to play with the ball like throw, catch and hold.

Voice chat feature has also been enabled in a walkie talkie manner. When the user presses the grip button on the controller, they will be able to send the voice over the network and everyone will be able to hear them.

In short, this project is like a game meeting room which can be played with different HMD’s and played in different platform like Windows and Mac OS.
